# Hyper Clicker

This is a modern cool looking auto clicker created with Tauri 2.0. 


## Used

- [Tauri](https://tauri.app/)
- [Tailwind](https://tailwindcss.com/)
- [DaisyUI](https://daisyui.com/)
- [Tauri-Plugin-Store](https://github.com/tauri-apps/plugins-workspace/tree/v1/plugins/store)
- [Enigo](https://github.com/enigo-rs/enigo)
- [device_query](https://github.com/ostrosco/device_query)
- [worker-timers](https://github.com/chrisguttandin/worker-timers)
